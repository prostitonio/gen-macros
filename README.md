> 🛠 **Мини-десктопное приложение для автоматической генерации макросов установки Xeuse 3.0.** 
> Забудьте о рутине и ручном вводе — автоматизируйте процесс за пару кликов!

---

## 📖 О проекте

Этот проект представляет собой легковесное десктопное приложение, написанное полностью на Python. Его главная цель — избавить вас от необходимости вручную писать и настраивать макросы для установки **Xeuse 3.0**. 

Приложение берет на себя всю рутину: вы выбираете нужные параметры в удобном интерфейсе, а программа генерирует готовый, валидный макрос, который можно сразу использовать.

### ✨ Возможности
- 🖥️ **Интуитивный GUI**: Удобный десктопный интерфейс (без необходимости лезть в консоль).
- ⚡ **Мгновенная генерация**: Создание макросов за доли секунды.
- 🎯 **Гибкая настройка**: Подстройка параметров установки Xeuse 3.0 под ваши нужды.
- 💾 **Экспорт**: Сохранение сгенерированных макросов в файл или копирование в буфер обмена одним кликом.

## 📦 Установка и запуск

### 1. Создание окружения
```bash
python3 -m venv envXeuseGenMac
source envXeuseGenMac/bin/activate
```
### 2. Установка библиотек
```bash
pip install PyQt5 numpy pyqtgraph pandas spec2nexus
```
### 3. Клонирование репозитория
```bash
git clone https://github.com/prostitonio/gen-macros.git
cd gen-macros
```

### 4. Запуск
```bash
python3 main.py
```
